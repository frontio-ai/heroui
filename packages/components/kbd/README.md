# @vezham/kbd

Keyboard key or kbd is a component to display which key or combination of keys performs a given action.

Please refer to the [documentation](https://heroui.com/docs/components/kbd) for more information.

## Installation

```sh
yarn add @vezham/kbd
# or
npm i @vezham/kbd
```
