# @vezham/badge

Badges are used as a small numerical value or status descriptor for UI elements.

Please refer to the [documentation](https://heroui.com/docs/components/badge) for more information.

## Installation

```sh
yarn add @vezham/badge
# or
npm i @vezham/badge
```
